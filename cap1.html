    • Richiami C++:

      • Namespace
        • per non inquinare il namespace globale, si usano altri namespace piu' specifici
        • operatore di scooping -> ::
        • keyword using
        • usati per aver lo stesso nome con piu' significati, a seconda del namespace
      • Argomenti di Default
        • non si puo' avere un argomento di default seguito da un argomento non di default
      • Puntatori a Funzioni
        • contiene indirizzo di memoria di una funzione, con segnatura e tipo di ritorno
        • esempio: double (*pf) (double); (double) indica il tipo di ritorno
        • pf=&sin; (*pf)(0); &sin indica la segnatura(sin) della funzione, mentre (0) da il valore per il parametro richiesto
      • keyword typedef
        • permette di dichiarare un alias per un tipo esistente
      • operator virgola ,
        • o sequenza
        • valuta le espressioni da sinistra verso destra e ritorna il valore dell'ultima espressione
      • type string
        • in C String è un array di caratteri e ha il arattere finale nullo '/0'(ASCII 0); questo è fonte di errori per il dover gestire puntatori di char.
        • in C++ String è un' istanziazione di template di classe della libreria STL.
        • bisogna includere il file header di class associato: #include e il namespace std: using namespace std;
        • nella class string in C++ ci sono un sacco di funzioni aggiuntive(metodi di classe) come: size() , empty() , ... e una costante predefinita importante : string::npos, utile per vari controlli.
        • C vs C++: char* s="..."; vs string s="...";
